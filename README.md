# k8s-Config-changer

### KKC - k8s Config Changer
is a tool to fast change k8s config file in the CLI

![img.png](img.png)


## Setup
To set it up by crateing a ``configs`` folder in ``.kube`` folder

```
.kube
  ├── config
  └── configs/
        ├── config.ikt449
        └── config.pingu

```
